# OS Scheduling Simulator in C

This repository contains an Operating Systems project implemented in C, focused on simulating CPU scheduling and resource allocation behavior.

## Overview
The project models process execution using queues, CPU bursts, and resource requests. It simulates how processes are scheduled and how resources are assigned, delayed, or released during execution.

## Features
- Process queue management
- CPU scheduling simulation
- Resource request handling
- Waiting state handling when resources are unavailable
- Test cases and supporting documentation included
